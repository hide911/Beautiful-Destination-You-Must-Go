# Beautiful-Destination-You-Must-Go

A simple and beautiful website showcasing amazing travel destinations around the world.

## Features

- Clean and simple design
- Responsive - works on all devices (mobile, tablet, desktop)
- Category filters (Beaches, Mountains, Cities, Nature)
- Beautiful destination cards with images and ratings
- Interactive interface with smooth animations

## Destinations Included

### Beaches
- Bali, Indonesia
- Maldives
- Santorini, Greece

### Mountains
- Swiss Alps, Switzerland
- Rocky Mountains, Canada

### Cities
- Paris, France
- Tokyo, Japan
- New York City, USA

### Nature
- Amazon Rainforest, Brazil
- Great Barrier Reef, Australia

## How to Use

1. Open `index.html` in your web browser
2. Browse all destinations or filter by category
3. Click "Learn More" button to explore destination details
4. Get inspired for your next trip!
